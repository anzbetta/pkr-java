# Frontend для системи "Факультатив"

## Встановлення

```bash
cd frontend
npm install
```

## Запуск

```bash
npm run dev
```

Застосунок буде доступний за адресою http://localhost:3000

## Збірка

```bash
npm run build
```

## Технології

- React 18
- TypeScript
- Vite
- React Router
- TanStack Query
- Zustand
- Tailwind CSS
